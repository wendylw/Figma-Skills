# Figma 常用快捷键

#### Frame

在Frame操作下，Figma提供默认的各种设备版本的Frame尺寸

**Frame 快捷键：**F

<figure><img src=".gitbook/assets/Frame-Default.png" alt=""><figcaption></figcaption></figure>

#### Group

需要把多个元素放在一个组里面

**Group 快捷键：**Command + G

#### 文本

**文本 快捷键：**T

#### 缩放

**缩放 快捷键：**k

**已经在缩放状态，撤回到只是移动状态：**ESC

#### 复制+拖拽

选中对象（需复制且拖拽出来的）+ （Mac: OPT / Windows: Alt) 直接拖就能实现
